Recipe Reference — DriftConditions

Structure

A recipe is a JSON5 object (comments allowed). Tracks play simultaneously; clips within a track play one after another.

{
  tracks: [
    {
      label: "my track",   // optional name (used by duck)
      volume: 100,            // 0–100, default 100
      effects: [ ],
      clips: [
        {
          classification: [ 'ambient' ],
          tags: [ 'drone' ],
          length: [ 'long' ],
          volume: 100,            // 0–100, default 100
          effects: [ ],
        }
      ]
    },
  ]
}

Classifications

Use lowercase values in recipes. Apply as few as accurately describe the clip.

ValueDescription
ambientDrone, ambient music, minimalist or bed-style audio
atmosphericField recordings, city ambiance, cafe, crowd noise
environmentalNatural sounds — thunder, birdsong, rain, ocean
premixedAlready-assembled mixes — radio excerpts, podcast clips
soundscapeComplex audio environments, theatrical sound design
archivalHistorical recordings, old radio, vintage speeches

narrativeRadio dramas, storytelling, narrative podcasts
instructionalEducational audio — lessons, how-tos, training
vocalmusicMusic with prominent vocals
instrumentalMusic without vocals — classical, jazz, ambient instrumental
experimentalAvant-garde, noise music, unconventional compositions
digitalSynthesizer, electronic beats, computer-generated sounds
effectClips for layering — stingers, transitions, radio signals, noise, foley
otherAnything that doesn't fit another category

Track Effects

Applied to the whole track.

Duration / structure

trim This track's duration sets the mix length first The first track in the recipe determines the mix length shortest Mix ends when the shortest finite track ends longest Mix ends when the longest finite track ends (default behavior)

Looping

loop Loop clips to fill the mix duration loop(n) Loop exactly n times

Fade

crossfade Crossfade between clips on this track fadeout Fade the track out at the end of the mix (default 3s) fadeout(n) Fade out over n seconds

Level / normalization

norm Normalize to default level norm(voice) Optimize for speech (foreground level) norm(music) Optimize for foreground music norm(bed) Background music bed level (quieter)

Color / texture

backward Reverse playback faraway Low-pass filter + reverb — sounds distant faraway(vol) Same, plus volume reduced to 30% telephone Narrow bandpass — telephone or AM radio effect

Volume modulation

wave() Smooth volume oscillation — see Wave Modifiers below

Routing

duck(label) Lowers this track's volume when the named track is playing — keeps a music bed from competing with a narrator

Wave Modifiers

wave() modulates volume with a smooth coherent-noise function. Parameters can be combined in any order.

Presets (speed)

wave() Default — active, noticeable oscillation wave(fast) Rapid, more agitated oscillation wave(slow) Gentler, longer period wave(slower) Very slow, barely perceptible movement

Modifiers (shape)

counter Invert the wave — loud when base is quiet (aliases: inverse, invert, inverted) soft Subtle amplitude — wave stays in upper range, less dramatic lifted Raised floor — wave never drops to silence bridge Peaks at crossings of normal and inverted wave (alias: transition)

Combining

Preset and modifiers can be combined in any order, comma-separated:

wave(slow, counter) Slow + inverted wave(slow, soft) Slow + subtle wave(lifted, counter) Raised floor + inverted wave(slow, lifted, counter) All three combined

Example Recipe

Spoken narrative over a looping ambient bed, with sidechain ducking:

{
  tracks: [
    {
      // Ambient drone bed — loops, fades out, ducks when narrator speaks
      label: "bed",
      volume: 60,
      effects: [ "loop", "norm(bed)", "wave(slow, soft)", "duck(narrator)" ],
      clips: [
        {
          classification: [ 'ambient' ],
          tags: [ 'drone', 'minimalist', 'sustained', 'atmospheric' ],
          length: [ 'long', 'huge' ],
        }
      ]
    },
    {
      // Spoken word — sets mix duration
      label: "narrator",
      volume: 100,
      effects: [ "trim" ],
      clips: [
        {
          classification: "silence",
          length: [ 'short' ]
        },
        {
          classification: [ 'narrative', 'spoken' ],
          tags: [ 'story', 'storytelling', 'reading', 'monologue' ],
          length: [ 'medium', 'long' ],
          effects: [ "norm(voice)" ],
        },
        {
          classification: "silence",
          length: [ 'short' ]
        }
      ]
    }
  ]
}
